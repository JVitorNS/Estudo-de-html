Parágrafos e quebras de linhas

Utilzamos da tag P. Assim, cpodemos criar parágrafos em nosso texto.

Para ocorrer a quebra de linha, temos que realizar atráves de outra tag. <b>.

Vale lembrar.

Para utilizarmos de tags sem ativá-las, temos as opções de alguns métodos, como por exemplo: 1) utilizar a opção de comentários. sendo possivel a sua visualização somente por onde o código foi feito, como no aplicativo Visual studio code. exemplo: 1.5) 2) Podemos deixar algumas tags de forma maiúscula.

Adição de emojis ao texto

Para isso, podemos copiar/colar, ou adicionar o codígo atráves do 💖. Fazemos ao adicionar &#x e o código do emoji, visto no site>

Imagens

O uso de imagens requer certa atenção, uma vez que imagens podem ou não conter diferentes tipos de diereitos autorais. Quando isso está feito, temos que pensar em como nosso site irá carregar essa imagem. Não desejamos um site lento, então é ideal reduzir o tamanho do arquivo. Usamos a tag <img>.

Adição de links e favicon

Para adicionarmos links e mudarmos o favicon, uzamos a tags <link>.
Vale lembrar que existem diferentes opções de favicon, desde criá-los em sites espefificos ou até mesmo em situações - como o de estudo - em que procuramos ícones do google para usarmos.

Onde colocar o link do favicon?

Deve-se colocar no heart(cabeça do texto).
